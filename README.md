# Conway-s-Game-of-Life-in-C
Simple terminal appllication in C for the conway's game of life

![image](https://user-images.githubusercontent.com/77921296/224089679-15b3cbad-b65c-487b-94f2-7af5bdccfcf4.png)

# Compiling
Move into the folder with a terminal and run the following:
```
cmake --build .
```
Afterwards you should be good to go!
